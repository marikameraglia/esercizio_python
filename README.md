# Esercitazione Python e SQLite

Questo progetto è un semplice esercizio in Python per imparare a gestire un database SQLite.

## Cosa fa il programma
Il file `main.py` permette di:
* Creare automaticamente un database chiamato `esercitazione.db`.
* Creare una tabella per i contatti.
* Inserire nuovi contatti (nome e telefono) tramite terminale.

## Come utilizzarlo
1. Assicurati di avere Python installato.
2. Apri il terminale nella cartella del progetto.
3. Esegui il comando:
   ```bash
   python main.py
