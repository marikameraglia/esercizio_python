# Esercitazione Finale: Gestione Classe Studente
Questo repository contiene lo script Python realizzato per l'esercitazione finale.

## Descrizione del Progetto
Il software implementa una classe chiamata `Studente` per la gestione dei dati accademici. Ogni oggetto creato rappresenta uno studente specifico con i propri attributi e metodi.

## Funzionalità
Il file `main.py` esegue le seguenti operazioni:
- **Definizione della classe**: Utilizza il metodo `__init__` per inizializzare nome, cognome, età e matricola.
- **Presentazione**: Un metodo che restituisce una stringa con i dati identificativi.
- **Gestione Voti**: Possibilità di aggiungere voti a una lista dinamica.
- **Calcolo Media**: Calcola automaticamente la media aritmetica dei voti inseriti.
- **Attività**: Un metodo che simula le ore di studio effettuate.

## Istruzioni per l'uso
Per testare il programma, eseguire il comando:
```bash
python main.py
